DK's Mini Games

Welcome to DK's Mini Games!

DK's Mini Games is a collection of fun and engaging browser-based games designed to provide entertainment and challenge for players of all skill levels. Our games are built with HTML5, CSS3, and JavaScript, ensuring a smooth and responsive experience across all devices.

Game Categories

Easy Level Games

  • Tic Tac Toe - Classic strategy game
  • Rock Paper Scissors - Quick decision making
  • Guess the Number - Simple number guessing game

Medium Level Games

  • Memory Card - Test your memory skills
  • Whack a Mole - Quick reflexes required

Hard Level Games

  • Snake Game - Classic arcade challenge

DK's Mini Games was created by Wai Yan Phyo Oo as a personal project to showcase web development skills and provide entertainment.
